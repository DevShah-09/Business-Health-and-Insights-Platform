# Business Health and Insights Platform

A comprehensive, full-stack financial analytics platform tailored for small businesses to monitor real-time transactions, track cash flow, and manage overall performance.

**🔴 Live Demo:** : https://business-health-and-insights-platfo.vercel.app/

## 📖 Project Overview

• Built interactive React and Tailwind dashboards to visualize key performance indicators (KPIs) and profit/loss metrics.
• Engineered high-performance backend services using FastAPI to power AI-driven predictive forecasting, scenario simulations, and automated anomaly detection.
• Integrated a customizable alert system and a Hinglish-capable AI chat assistant for intuitive financial querying and real-time notifications.
• Designed specifically for small businesses to proactively manage risks and explore future outcomes through real-time data processing.

## ✨ Key Features

* **Real-Time Financial Dashboards:** Interactive charts tracking cash flow, profit and loss, and essential business Key Performance Indicators (KPIs).
* **Predictive Forecasting:** Utilizes machine learning to forecast future income and expenses based on historical data.
* **Automated Anomaly Detection:** Statistically flags unusual transactions (e.g., unexpected spikes in expenses) to prevent fraud and errors.
* **Scenario Simulation Engine:** Allows business owners to simulate financial decisions and predict cascading impacts on their business stability.
* **Smart AI Chat Assistant:** An embedded NLP chatbot capable of understanding and communicating in **Hinglish**, enabling users to intuitively query their financial data (e.g., "Aapke marketing expenses kaafi high hain").
* **Customizable Alerts:** Real-time push notifications and alerts for critical business events or metric thresholds.

## 🛠️ Tech Stack

**Frontend:**
* React.js
* Tailwind CSS
* JavaScript (Vite)

**Backend:**
* Python
* FastAPI
* RESTful APIs

**Analytics & Machine Learning:**
* **Linear Regression:** For predictive financial forecasting with bounded confidence intervals.
* **LSTM (Long Short-Term Memory):** Neural networks for scenario simulation and systemic risk evaluation.
* **Z-Score Statistical Analysis:** Core algorithm for detecting data anomalies.
* **LLMs (GPT-3.5 / Gemini-2.5-Flash):** Powers the conversational AI chat assistant.
